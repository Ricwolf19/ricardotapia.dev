## Contexto

Chachitos es una empresa mexicana con más de 70 años produciendo cereales de trigo inflado. El proyecto es su sitio web oficial: una presencia moderna, rápida y responsive.

## Arquitectura

Sitio Next.js con App Router organizado en rutas claras (Home, Nosotros, Sabores, Menú, Encuéntranos):

- `components/` — componentes reutilizables, incluida la integración de Google Maps y el localizador de tiendas.
- `const/` — constantes institucionales (valores, direcciones, redes sociales).
- `mocks/` — datos de ejemplo (productos y sabores).
- `utils/` — utilidades como el cálculo de distancia con la fórmula de Haversine.

## Stack

- Next.js 15 + React 19 + Tailwind CSS + TypeScript.
- Google Maps API para el localizador de tiendas (con marker clustering).
- reCAPTCHA y Resend para los formularios de contacto.
- scrollreveal para animaciones de entrada.

## Resultado

Un sitio institucional con un localizador de tiendas funcional que calcula la distancia del usuario a cada punto de venta. [Ver sitio](https://webpage-indol-five.vercel.app/).
