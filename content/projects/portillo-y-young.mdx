## Contexto

Sitio web corporativo para Portillo y Young. El objetivo fue una presencia profesional, multilingüe, con portafolio de proyectos y captación de contactos.

## Arquitectura

Aplicación Next.js con App Router e internacionalización por segmento de ruta: todas las páginas viven bajo `app/[lang]/` (home, about, services, contact y portfolio con detalle `[slug]`). La lógica de formularios se implementa como server action en `actions/forms.ts`.

## Stack

- Next.js 15 + React 19 + Tailwind CSS + TypeScript.
- next-safe-action para server actions tipadas.
- React Hook Form + Zod para validación de formularios.
- Resend + React Email para correo. reCAPTCHA para protección anti-spam.

## Resultado

Un sitio multilingüe con portafolio navegable por proyecto y formularios de contacto validados y protegidos, listo para crecer en nuevos idiomas gracias al enrutamiento por `[lang]`. [Ver sitio](https://portilloyyoung.com/en).
