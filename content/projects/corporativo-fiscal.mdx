## Contexto

Corporativo Fiscal opera una suite de servicios fiscales y legales que requería digitalizarse en una plataforma única, con módulos especializados por área y un sistema de presentaciones colaborativas en tiempo real.

## Arquitectura

Arquitectura de microservicios sobre Express con 8 aplicaciones privadas: Legal, Clientes, Materialización, Recepción, RH, Admin, Landiabar y Visión Fiscal. El frontend usa React + Vite y la persistencia MongoDB.

El módulo de presentaciones live combina **WebSockets** para tiempo real, **Excalidraw** para pizarra colaborativa y **YJS** (CRDT) para resolución de conflictos sin servidor de verdad central.
